工具整理

Mac 软件

  • Input Source Pro 一个输入法自动切换工具，用了好久了，使用起来很默默无闻，但是帮我节省了很多时间。
  • Raycast 这个全局搜索工具超级好用和强大，假如你还在用 Alfred，很推荐你试试，此外他的出现，通过插件还可以帮你干掉很多软件。
  • Mos：一个平滑你的鼠标滚动效果或单独设置滚动方向的小工具，让你的滚轮爽如触控板（开源）

开发输出工具

  • JetBrains 全家桶，我是一个重度用户，推荐你使用，他们的 IDE 都很好用。
  • VSCode
  • Godot 一个开源的游戏引擎，我用它写了一些小游戏，非常好用。
  • Obsidian 一个非常好用的笔记工具，支持 Markdown，支持插件，支持本地存储，支持同步，支持自定义 CSS，支持插件，支持插件，支持插件，重要的事情说三遍。
  • Another Desktop Manager 一个 Redis 客户端工具。
  • Postman 一个 API 测试工具，可以帮助你测试 API 接口。
  • SwitchHosts 一个方便的 hosts 管理工具。

系统管理

  • btop 使用介绍 一个类似于 htop 的 Mac 系统监控工具，可以查看系统的 CPU、内存、磁盘、网络等信息。
  • Onefetch 一个命令行工具，可以显示 Git 仓库的详细统计信息，比如代码行数、体积、贡献者等等。
  • cakebrew： 如果使用 Homebrew 的话，可以使用这款来进行管理（开源）
  • Tunnelblick：OpenVPN 客户端（开源）
  • 腾讯柠檬：腾讯出品的系统清理工具（开源）
  • n8n

web网页工具

  • 数据可视化目录
  • 示意图利器 D2
  • lucide-icons社区制作的美观且一致的图标工具包。开源项目和Feather Icons的一个分支。

阅读

  • Readest 一款开源电子书阅读器，跨平台桌面应用，基于 Next.js 15和 Tauri v2。
